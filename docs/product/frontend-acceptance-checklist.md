# 前端原型验收清单

## 1. 页面清单

| 页面 | 路由 | 状态 | 说明 |
|------|------|------|------|
| 首页 Dashboard | `/` | ✅ | 家庭总资产/今日收益/累计收益/持仓/已实现/现金/图表/排行 |
| 成员列表 | `/members` | ✅ | 3个成员卡片，点击进入详情 |
| 成员详情 | `/members/[id]` | ✅ | 摘要/持仓/已清仓/交易/投资理念/趋势 |
| 持仓列表 | `/holdings` | ✅ | 按成员分组，含已清仓 |
| 单仓详情 | `/holdings/[id]` | ✅ | 摘要/价格走势/买卖点/交易生命周期/新闻/AI建议 |
| 导入页 | `/import` | ✅ | 导入中心：截图OCR/手动持仓/手动交易/批量粘贴，4种导入方式 |
| 简报页 | `/brief` | ✅ | 市场概览/家庭影响/成员影响/新闻/风险/建议/推送/历史 |

- [x] 累计收益 = 持仓收益 + 已实现收益
- [x] 所有"收益"字段标注了具体类型（今日/持仓/已实现/累计/区间）
- [x] 所有金额有币种符号和千分位
- [x] 正负数有 +/-
- [x] 已清仓标的市值和持仓收益为0
- [x] 已清仓标的保留已实现收益和累计收益
- [x] 成员层级 = 该成员所有单仓和现金的聚合
